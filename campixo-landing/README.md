# Campixo Landing Page

Static landing page for `campixo.com` / `www.campixo.com`.

## Links

- Live school portal CTA: `https://ips.campixo.com/`
- Instagram: `https://www.instagram.com/campixo`
- Founder LinkedIn: `https://www.linkedin.com/in/kanhaiya-dubey/`
- Logo asset: `campixo-logo.jpg`, downloaded from the Campixo Instagram profile image

## Deployment

Deploy this folder to Cloudflare Pages as a static site:

- Build command: leave empty
- Build output directory: `/`

Recommended DNS setup:

- `campixo.com` -> Cloudflare Pages landing page
- `www.campixo.com` -> Cloudflare Pages landing page
- `*.campixo.com` -> Railway school portal app

If you update the official Campixo logo later, replace `campixo-logo.jpg` with the new file using the same filename.
