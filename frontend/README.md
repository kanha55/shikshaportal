# Shiksha Portal — Frontend (React)

React SPA for school admin, student dashboards, and public school pages.

## Planned setup (Sprint 1)

- Vite + React + TypeScript (or JavaScript)
- Tailwind CSS
- `react-i18next` — Hindi (default) + English UI
- Noto Sans Devanagari font
- Axios + JWT auth context
- Subdomain-aware API base URL

## Scaffold command

```bash
npm create vite@latest . -- --template react
```

See root `README.md` and `docs/shiksha_portal_roadmap.xlsx` for full sprint plan.
